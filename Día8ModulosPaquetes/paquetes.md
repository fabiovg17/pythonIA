en CMD se puede verificar la versión de Python instalada usando el comando `python --version` o `python -V`.


para instalar paquetes de Python se puede usar el comando `pip install nombre_del_paquete`.

para verificar los paquetes instalados se puede usar el comando `pip list` o `pip freeze`.
para desinstalar un paquete se puede usar el comando `pip uninstall nombre_del_paquete`.
para crear un entorno virtual se puede usar el comando `python -m venv nombre_del_entorno`.
para activar el entorno virtual en Windows se puede usar el comando `nombre_del_entorno\Scripts\activate` y en Mac o Linux se puede usar el comando `source nombre_del_entorno/bin/activate`.
